# Product Drop Checklist

Use this checklist for every free or paid product launch.

## Product Definition

- [ ] Product name confirmed
- [ ] Customer problem stated in one sentence
- [ ] Who it is for defined
- [ ] Who it is not for defined
- [ ] Deliverables listed
- [ ] Price confirmed
- [ ] Refund/support policy confirmed
- [ ] Fitness disclaimer added

## Asset Creation

- [ ] Product PDF or files complete
- [ ] Cover image or mockup complete
- [ ] Checkout product description written
- [ ] Delivery email written
- [ ] FAQ written
- [ ] Upsell or next-step email written
- [ ] Testimonial request written

## Funnel Setup

- [ ] Landing page live
- [ ] Checkout link live
- [ ] Payment test completed
- [ ] File delivery tested
- [ ] Email tags tested
- [ ] Welcome/onboarding sequence tested
- [ ] Support inbox link visible
- [ ] UTM links created

## TikTok Launch Assets

- [ ] Launch announcement video
- [ ] Product walkthrough video
- [ ] Objection-handling video
- [ ] Founder story video
- [ ] FAQ video
- [ ] Comment reply video prompts
- [ ] Pinned comment CTA written

## Email Assets

- [ ] Waitlist email
- [ ] Launch email
- [ ] Reminder email
- [ ] FAQ email
- [ ] Last-call email if using a deadline
- [ ] Buyer onboarding email
- [ ] Feedback/testimonial email

## Automation Checks

- [ ] New signup tag works
- [ ] New buyer tag works
- [ ] Delivery automation works
- [ ] Dashboard row updates
- [ ] Support triage route works
- [ ] GitHub product issue created
- [ ] Failed-payment/refund process documented

## Launch-Day Runbook

- [ ] Post launch TikTok
- [ ] Send launch email
- [ ] Check checkout works after first click
- [ ] Reply to comments in first hour
- [ ] Record all questions in GitHub
- [ ] Create follow-up videos from questions
- [ ] Check sales and email metrics twice

## Post-Launch Review

- [ ] Total views
- [ ] Profile visits
- [ ] Bio clicks
- [ ] Leads
- [ ] Sales
- [ ] Conversion rate
- [ ] Refunds
- [ ] Support tickets
- [ ] Top objections
- [ ] Product improvements
- [ ] Next experiment
